\# Founder Voice Engine



An AI tool that analyzes a founder's LinkedIn writing style and generates new posts that match their voice.



\## What it does

\- Paste 2-5 LinkedIn posts from any founder

\- AI analyzes their tone, sentence structure, hooks, and patterns

\- Generates new posts on any topic in their exact voice



\## Why I built this

I noticed most AI-generated LinkedIn content sounds generic. This tool solves that by learning individual writing patterns before generating content.



\## How to run

1\. Get a free Groq API key at console.groq.com

2\. Set your key: `$env:GROQ\_API\_KEY="your-key"`

3\. Run: `python app.py`



\## Tech Stack

Python, Groq API, LLaMA 3.3
